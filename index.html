How to make it an Android / iOS app?

  1. Zip up the www folder + config.xml
  2. Upload to Phonegap Build Service
  3. Get the APK / ipa file
  4. Profit :)

How to make it an Android / iOS app? Alternative

        $ ionic add platform android
        $ ionic build

How to make it a Firefox OS app?

  1. Zip up the www folder + manifest.webapp file
  2. Upload to Firefox Marketplace
